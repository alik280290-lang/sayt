Введение

Что такое reference board

Reference board — это структурированный визуальный документ, который ты скармливаешь AI-модели как единственный источник правды. Один борд держит лицо персонажа, пропорции продукта, атмосферу локации — и каждая следующая генерация остаётся консистентной.

Это решает главную боль AI-видео — то, что в каждом новом кадре персонаж может стать другим. Лицо съезжает, одежда меняется, рост гуляет. Reference board фиксирует всё в одном документе и заставляет модель ему следовать.

В гайде — 6 типов бордов: Character, Object, Pose, Location, Shot, Creature. Каждый берёт одну картинку и превращает её в полноценный production-документ.

Я гоняю эту систему каждый день — на кейсах для Kabrita, KGM, PROCAR. Когда нужно держать одного и того же персонажа через 20 сцен — нет другого способа.

Процесс

Как сгенерировать борд за 5 шагов

1
Готовишь исходник

Чистая, детальная картинка — портрет, продукт, локация. Чем лучше исходник, тем точнее борд. Тёмная мутная фотка → борд развалится.

2
Загружаешь в Higgsfield или PicsArt Flow

Higgsfield: Create → Image. PicsArt: Create → Workflows → Image Node → AI Image Node.

3
Выбираешь модель GPT Image 2

Она особенно хорошо читает плотные визуальные лейауты — другие модели хуже держат структуру борда.

4
Копируешь промпт нужного борда

Каждый промпт ниже. Не сокращай — каждая деталь говорит модели какие именно панели генерировать.

5
Жмёшь Generate

Получаешь развёрнутый production-document. Если первая генерация кривая — перегенерируй. GPT Image 2 даёт вариативность.

Советы

Pro tips от практика

  • Источник решает всё. Портретные или квадратные кропы работают лучше для персонажей и существ. Широкоугольные кадры — для локаций.
  • Перегенерируй при кривом результате. GPT Image 2 даёт вариативность, лейаут может слегка плавать между прогонами.
  • Стакай борды. Сначала Character Board, потом его результат как исходник для Pose Board — двойная консистентность.
  • Shot Board — самое мощное. Один кейфрейм → готовая 12-шотовая раскадровка для Seedance.
  • Все подписи на бордах — на английском. Не переводи, GPT Image 2 хуже следует русскому в плотной типографике.
  • Рекламу для русского рынка гони с русскими надписями в финальных кадрах, но борд держи на английском как референс структуры.

01

Character Board

Когда использовать

Любой человек, портрет, AI-инфлюенсер. Перед AI-видео с одним героем — всегда сначала Character, потом Pose, потом Shot.

Что внутри:

  • Hero portrait + метадата: имя, возраст, рост, телосложение, волосы, глаза, аутфит, характер
  • Panel 01 — Views: Front · 3/4 Left · Side Left · Back · 3/4 Right
  • Panel 02 — Expressions: Neutral · Smile · Thoughtful · Focused · Serious
  • Panel 03 — Details: face/eyes close-up · distinctive outfit detail
  • Panel 04 — Outfit flat-lays: jacket · shirt · trousers · watch · shoes
  • Panel 05 — Lighting: Soft Daylight · Warm Tungsten · Cool Blue Night · Hard Cinematic Side-Light
  • Panel 06 — Color palette: 6 HEX свотчей

The same person must appear in every panel — same age, same outfit. All on-image labels in ENGLISH. Object Board

Когда использовать

Любой объект, продукт, реквизит. Идеально для рекламы товара, где нужно держать его геометрию через всю серию кадров. Работает на чём угодно — от меча до автомобильного диска R20.

  • Panel 02 — Details (4 макро): handle/grip · body/blade · engraving/maker's mark · edge/wear
  • Panel 03 — Lighting/Mood: 4 сценария света
  • Panel 04 — Color palette: 6 HEX свотчей

Shot Board

Когда использовать

Когда у тебя есть одна сильная картинка и нужно превратить её в сцену. Берёт один кейфрейм и раскладывает его на готовую 12-шотовую раскадровку. Экономит часы раскадровки.

Самое мощное оружие AI-фильммейкера. Готово для прямой передачи в Seedance.

Что внутри:

  • Source keyframe + метадата сцены: scene, act, runtime, setting, characters, emotional arc, camera style, color script
  • Panel 01 — Shot sequence (12 кадров): Establishing Wide → Character Enters → OTS → Close-Up → Insert → Counter OTS → Two-Shot → Tracking → Low Angle Wide → ECU Eyes → Slow Pull-Back → Wide Resolution
  • Panel 02 — Key emotional beats: 3 драматических close-up
  • Panel 03 — Camera notes: lens, motion, framing, height
  • Panel 04 — Lighting continuity: 4 кадра
  • Panel 05 — Coverage types: Wide · Medium · Close-Up · Insert · OTS
  • Panel 06 — Color palette: 6 HEX свотчей

Продвинутый уровень

Комбинации бордов

Реальная сила системы — когда стакаешь борды один в другой. Результат первого становится исходником для следующего, и консистентность копится.

Character → Pose

Сначала строишь полную character-bible. Потом её результат скармливаешь в Pose Board. Двойная консистентность лица и тела под анимацию.

Character → Shot

Один и тот же персонаж через целую кинематографическую сцену. Идеально для AI-короткометражек.

Location → Shot

Консистентность окружения, запеченная в раскадровку. Каждый из 12 кадров живёт в одной локации с одинаковыми правилами света.

Creature → Pose

Не-человеческий персонаж под анимацию в Seedance с полным behavioral референсом.

Object → Shot

Проп получает собственное кинематографическое раскрытие. Полезно для рекламных тизеров продукта.

Финальный шаг

Интеграция с Seedance

Когда борд готов, его 12-шотовая последовательность (Shot Board) переходит прямо в видео-генерацию.

1
Делаю Character Board из портрета

Исходник — чистый портрет, желательно нейтральный фон.

2
Его результат → исходник для Shot Board

Character Board стал источником правды. Теперь Shot Board использует его как кейфрейм.

3
Каждый из 12 шотов → кейфрейм в Seedance

Shot Board даёт 12 готовых кадров. Каждый идёт в Seedance как отдельный кейфрейм.

4
В промпте Seedance — @Image1 несколько раз

Это усиливает привязку к референсу и держит консистентность через весь клип.

5
Цветокор и монтаж

Cold blue night + warm amber lights, монтаж staccato без dissolves. Так я делал кейс Kabrita для VIDMK — взял победу в номинации Best AI Realism.
